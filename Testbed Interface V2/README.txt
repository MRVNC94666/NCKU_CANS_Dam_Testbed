[Operate illustration]

"SCAN"" buttom : 
Press to load the status of the dam and start reading status every sec.

"OPERATE" buttom : 
Press to start the operation.

"UP" buttons : 
Press to send command to PLCs to lift the Gate.
Turn red while the gate is going up.

"DOWN" buttons : 
Press to send command to PLCs to close the Gate.
Turn red while the gate is going down.

"STOP" buttons : 
Press to send command to PLCs to stop the Gate.
Turn red while the gate is still.
